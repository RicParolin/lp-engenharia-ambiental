# 🌿 Landing Page - Felipe Di Mello | Engenharia Ambiental

Uma Landing Page de alta performance e conversão desenvolvida para um Estrategista Industrial focado em Engenharia Ambiental. O projeto foi concebido para transmitir autoridade técnica, segurança regulatória e eficiência corporativa para diretores e conselhos de administração (B2B).

Este projeto é uma entrega *End-to-End* (de ponta a ponta), englobando desde a estruturação do código até a criação da identidade visual e curadoria de imagens com Inteligência Artificial.

## 🚀 Escopo do Projeto e Tecnologias

Além do desenvolvimento Front-end, este case destaca a integração de ferramentas de IA no fluxo de trabalho de design e desenvolvimento:

* **Desenvolvimento Front-end:**
  * **HTML5:** Semântica rigorosa para otimização de SEO e acessibilidade.
  * **CSS3:** Design responsivo *Mobile-First*, custom properties (variáveis) para consistência de marca e animações fluidas.
  * **JavaScript (Vanilla):** Lógica assíncrona, validação de formulários em tempo real e uso da API `IntersectionObserver` para animações baseadas em scroll (Scroll Reveal e Contadores).
* **Design & Identidade Visual:**
  * **Criação de Logos:** Design e vetorização da identidade visual do cliente.
  * **Prompt Engineering (Gemini Pro):** Geração, tratamento e curadoria de imagens de alta fidelidade e adequação de paleta de cores através de prompts avançados, otimizando o tempo de entrega e elevando a qualidade visual.

## 💡 Destaques Técnicos

1. **Performance e Código Limpo:** O projeto foi construído sem dependência de bibliotecas ou frameworks externos (como jQuery ou bibliotecas de animação pesadas), garantindo um carregamento ultrarrápido, essencial para a retenção de usuários.
2. **Engenharia de Prompt:** Uso estratégico de IA Generativa para compor o visual da página, provando capacidade de adaptação às novas ferramentas do mercado de tecnologia.
3. **UX/UI Focado em Negócios:** A arquitetura da informação foi pensada para guiar o olhar do visitante corporativo através de dados de mercado, culminando em um CTA (Call to Action) direto para o WhatsApp do especialista.
